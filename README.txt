Projet ZOO

## Description

Bienvenue dans le Zoo fantastique, un jeu qui met en scène des animaux mythiques. 
Dans le jeu vous serez le directeur et vous allez devoir prendre soin des animaux.

Prés requis:
Assurez-vous d'avoir java installez sur votre machine comme par exemple sur:
    Eclipse
    VSCode
    ou autre capable de compiler du java.

## Installation
1. Clonez ce dépôt : git clone https://lien-vers-votre-repo.git
2. Naviguez vers le dossier du projet : cd nom-du-projet
3. Compilez le code : javac game.java
4. Exécutez le jeu : java game

Guide du joueur :
    Lorsque le jeu ce lance choisissez un animal parmis ceux disponibles comme le Dragon
    Puis explorer les différentes fonctionalités du jeu en intéragissant avec les animaux,
    Vous pourriez découvrir leur caractéristique unique.

## Utilisation
- #Dragon
  - manger: Fait manger tous les dragons.
  - son: Fait rugir tous les dragons.
  - soin: Soigne tous les dragons.
  - dormir: Fait dormir tous les dragons.
  - courrir: Fait courir tous les dragons.
  - nager: Fait nager tous les dragons.

- #Kraken
  - manger: Fait manger tous les krakens.
  - son: Fait sonner tous les krakens.
  - soin: Soigne tous les krakens.
  - dormir: Fait dormir tous les krakens.
  - nager: Fait nager tous les krakens.
  - pondre: Permet à deux krakens de se reproduire.

Cela fonctionne pour chaques créatures mythiques.

## Structure du Code
L'architecture du code est structuré en classes pour chaque types d'animal.
Les actions spécifique des different animaux mythique sont définies dans ces classes
